Instrukcja obsługi edytora poziomów do Superhoga:

  1. Skróty klawiszowe
    • ESCAPE - wyjście z programu
    • ALT + s - zapisz poziom
    • ALT + o - otwórz zapisany poziom
    • F2 - pauza
    • F3 - włącz/wyłącz tryb siatki
    • F4 - pokaż/ukryj okienko wyboru kafelków
    • F5 - włącz/wyłącz animację kafelków
    • q - Zaznacz fragment planszy
    • CTRL + c - skopiuj zaznaczone
    • CTRL + V - wklej zaznaczone
    • HOME - przejdź do lewej krawędzi planszy
    • CTRL + HOME - przejdź do górnej krawędzi planszy
    • END - przejdź do prawj krawędzi planszy
    • CTRL + END - przejdź do dolnej krawędzi planszy
    • ALT + HOME - przejdź do pozycji startowej gracza
  2. Mysza
    • Lewy przycsik - zmień zawartość kafelka
    • Prawy przycisk - przewijaj mapę
